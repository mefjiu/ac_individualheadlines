
# Custom Headlines h1 to Category
Module for Prestashop 1.7 - add basic Headlines H1  your Category


Override Category.tpl
Edit a file here /themes/[YOUR_THEME]/templates/catalog/listing/category.tpl and update Headlines H1:

<pre>
<h1 class="h1">{widget name='ac_individualheadlines'}</h1>
</pre>
